SISTEMA PESCD - UFSCar

Sistema web desenvolvido com Spring Boot para apoiar o fluxo de ofertas, inscrições, planos de trabalho, documentação e relatórios finais.

**User Story**

Abaixo estão listadas as user stories do sistema e seus respectivos responsáveis:


**Yuki Hiroji Oyama**

AD01 - COMO Administrador, EU QUERO gerenciar o cadastro de usuários, PARA manter usuários e definir acesso segundo perfis definidos - BAIXA

AL02 - COMO Aluno, EU QUERO enviar o plano de trabalho, PARA que o professor supervisor aprove o plano e inicie o estágio - ALTA 

S01 - COMO Secretário, EU QUERO criar uma oferta, PARA que o workflow do PESCD iniciar no semestre - ALTA 

U01 - COMO Administrador ou Secretário ou Aluno ou Professor Supervisor ou Professor Responsável, EU QUERO realizar o login, PARA ter acesso às funcionalidades únicas do meu perfil no sistema - ALTA
